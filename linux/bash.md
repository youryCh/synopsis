`clear` | `CTRL+L` - очистить терминал.

`exit` | `CTRL+D` - logout.

`vi <file>` - open file in Vi editor.

___

`CTRL+INSERT` - copy in bash.
`SHIFT+INSERT` - paste in bash.

___

`diff` - сравнить файлы.
  - `-bur` - подробности

```
// выведет диф в файл
diff dir1/file2 dir2/file2 > change.diff
```
